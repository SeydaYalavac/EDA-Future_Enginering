TR:
Keşifçi Veri Analizi ve Özellik Mühendisliği (Wine Quality)
Bu proje, veri bilimi süreçlerinin en kritik adımı olan veri hazırlama ve anlama aşamalarını içermektedir. Şarap kalitesini (WineQT) etkileyen parametrelerin derinlemesine incelenmesi hedeflenmiştir.

Neler Yapıldı?

Veri Tanıma: Veri setinin yapısal analizi, eksik verilerin kontrolü ve betimsel istatistikler.

Veri Görselleştirme: Değişkenler arasındaki korelasyonların heatmap ve scatter plotlar ile analizi.

Aykırı Değer (Outlier) Analizi: Model başarısını olumsuz etkileyebilecek değerlerin tespiti ve yönetimi.

Özellik Mühendisliği (Feature Engineering): Yeni değişken türetme ve veriyi makine öğrenmesi modellerine hazır hale getirme işlemleri.

ENG:
Exploratory Data Analysis & Feature Engineering (Wine Quality)
This project focuses on the most critical stages of the data science pipeline: data understanding and preparation. The primary goal is to perform an in-depth analysis of the parameters affecting wine quality (WineQT).

Key Features:

Data Exploration: Structural analysis of the dataset, missing value checks, and descriptive statistics.

Data Visualization: Analyzing correlations between variables using heatmaps and scatter plots.

Outlier Analysis: Identifying and managing anomalous data points that may affect model performance.

Feature Engineering: Creating new variables and preprocessing data to make it ready for machine learning algorithms.
